Yleisellä tasolla tietoturva on Linuxissa hyvällä tasolla Unixin rakenteen ja vapaiden ohjelmien ansiosta. On silti erittäin tärkeätä pitää huolta tietoturvan perusteista, joita ovat käyttäjätunnukset, salasanat, palomuuri ja virusskannerit. Suurimmat riskit tietoturvaan liittyvät kuitenkin palveluiden säätö- ja ohjelmointivirheisiin sekä selainten tietoturva-aukkoihin. Järjestelmän päivittäminen säännöllisesti pitää tietoturvaa yllä. Tietoturvaan saadaan lisävarmuutta käyttämällä palomuuria, TCP Wrapperia, SELinuxia tai IDSia, jotka ovat yleensä aina päällä.

Tietoturvapäivitykset

Kun päivityksiä on saatavilla, näkyy työpöydän ilmoitusalueella siitä kertova kuvake. Päivitysten asentaminen onnistuu yleensä napsauttamalla tätä kuvaketta tai käyttäen päivitysohjelmaa, joka löytyy valikon polusta Järjestelmä->Ylläpito->Päivitysten hallinta.

Päivityksen voi tehdä myös terminaalia käyttäen seuraavia vaiheita seuraamalla.

Päivitetään pakettitietokanta:

sudo aptitude update tai sudo apt update

Asennetaan päivitykset:

sudo aptitude upgrade tai sudo apt upgrade -y (-y asentaa päivitykset automaattisesti)

Päivitys uuteen versioon:

sudo do-release-upgrade tai sudo do-release-upgrade -d (jos haluaa uusimman kehittyneen version käyttöön, lisätään perään vipu -d)

Virusskannerit

Virusskanneria suositellaan käytettäväksi, jos samassa tietokoneessa on asennettuna Windows-käyttöjärjestelmä, jonka levyosiolla olevia tiedostoja Linuxilla käytetään tai Linux toimii palvelimena, jonka kautta Windows-koneilla on pääsy Internetiin. Vaikka virus tai haittaohjelma ei Linuxia haittaisikaan, se voi kulkeutua Linuxin kautta Windowsiin.

Linuxiin on saatavilla useita virusskanneria, joista tässä mainittakoon F-prot ja ClamAV. AVScan on Gnome-työpöytäympäristöön suunniteltu graafinen käyttöliittymä ClamAV-ohjelmaan.

ClamAV:n ja AVScanin asentaminen terminaalissa:

sudo apt-get update
sudo apt-get install clamav avscan

Ellei ohjelman pikakuva ilmesty ohjelmavalikkoihin, saat ohjelman käynnistymään kirjoittamalla komentokehotteeseen seuraavan komennon:

avscan

AVScan ei käynnisty itsenäisesti koneen käynnistyessä, vaan se täytyy ajaa erikseen joka kerta. Jos haluat päivittää virustietokannat, saatat joutua käynnistämään AVScanin sudo-komennolla terminaalissa:

sudo avscan

Vakoiluohjelmien etsijät (rootkit)

Asenna vakoiluohjelmien etsijät kirjoittamalla terminaaliin komento:

sudo apt-get install chkrootkit rkhunter

Käynnistä chkrootkit kirjoittamalla terminaaliin:

chkrootkit

Käynnistä Rootkithunter kirjoittamalla terminaalissa:

rkhunter -c

Yhteenveto

Asenna yhdellä terminaalikomennolla kaikki edellä mainitut ohjelmat.

sudo apt-get install clamav avscan chkrootkit rkhunter
